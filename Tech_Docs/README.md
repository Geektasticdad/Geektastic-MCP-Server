## At a glance

- **Stack:** TypeScript throughout. Node 22 + Express + `@modelcontextprotocol/sdk`
  (backend), React 18 + Vite + Tailwind (Web UI), PostgreSQL + Prisma (storage),
  Zod (validation everywhere), pnpm workspaces (monorepo).
- **Single deployable:** one Node container serves the React UI (static build),
  the `/api` management REST API, and the `/mcp` MCP endpoint, all on one port
  (8080). Paired with a Postgres container. See [Deployment](08-Deployment.md).
- **Pluggable connectors:** exactly one connector exists today
  (`geektastic-realms`), but the codebase is structured so a new application =
  a new module implementing `AppConnector` — see
  [Connector SDK](07-Connector-SDK.md).
- **Two auth systems, deliberately separate:** session-cookie auth for the Web
  UI (`/api/*`), and bearer-token auth for `/mcp` (either a static
  admin-issued token, or a short-lived OAuth 2.1 access token). See
  [Security](06-Security.md) and [OAuth 2.1](05-OAuth2.md).
